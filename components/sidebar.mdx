Shadcn Admin provides a flexible sidebar navigation system with support for collapsible navigation, team switching, nested menus, and responsive layouts.

## Overview

The sidebar system is located in `src/components/layout/` and includes:

- **AppSidebar** - Main sidebar container
- **NavGroup** - Grouped navigation items
- **TeamSwitcher** - Multi-team dropdown selector
- **NavUser** - User profile section
- **Sidebar Primitives** - Base components from `src/components/ui/sidebar.tsx`

## AppSidebar

The main sidebar component that wraps all navigation elements.

### Structure

From `src/components/layout/app-sidebar.tsx:15-37`:

```tsx
import {
  Sidebar,
  SidebarContent,
  SidebarFooter,
  SidebarHeader,
  SidebarRail,
} from '@/components/ui/sidebar'
import { AppSidebar } from './app-sidebar'
import { NavGroup } from './nav-group'
import { NavUser } from './nav-user'
import { TeamSwitcher } from './team-switcher'
import { sidebarData } from './data/sidebar-data'

export function AppSidebar() {
  const { collapsible, variant } = useLayout()
  
  return (
    <Sidebar collapsible={collapsible} variant={variant}>
      <SidebarHeader>
        <TeamSwitcher teams={sidebarData.teams} />
      </SidebarHeader>
      
      <SidebarContent>
        {sidebarData.navGroups.map((props) => (
          <NavGroup key={props.title} {...props} />
        ))}
      </SidebarContent>
      
      <SidebarFooter>
        <NavUser user={sidebarData.user} />
      </SidebarFooter>
      
      <SidebarRail />
    </Sidebar>
  )
}
```

### Sidebar Variants

The sidebar supports three variants:

- **sidebar** - Standard sidebar (default)
- **floating** - Floating sidebar with backdrop
- **inset** - Inset sidebar with padding

```tsx
import { Sidebar } from '@/components/ui/sidebar'

<Sidebar variant="floating" collapsible="icon">
  {/* Content */}
</Sidebar>
```

### Collapsible Modes

- **offcanvas** - Hide completely on mobile
- **icon** - Collapse to icon-only view
- **none** - No collapse behavior

## NavGroup

Organizes navigation items into labeled groups with support for nested items.

### Type Definitions

From `src/components/layout/types.ts:15-36`:

```tsx
type BaseNavItem = {
  title: string
  badge?: string
  icon?: React.ElementType
}

type NavLink = BaseNavItem & {
  url: LinkProps['to'] | (string & {})
  items?: never
}

type NavCollapsible = BaseNavItem & {
  items: (BaseNavItem & { url: LinkProps['to'] | (string & {}) })[]
  url?: never
}

type NavItem = NavCollapsible | NavLink

type NavGroup = {
  title: string
  items: NavItem[]
}
```

### Basic Usage

```tsx
import { NavGroup } from '@/components/layout/nav-group'

const navGroup = {

      url: '/users',
      icon: Users,
      badge: '12',
    },
    {
      title: 'Settings',
      url: '/settings',
      icon: Settings,
    },
  ],
}

<NavGroup {...navGroup} />
```

## Layout Integration

Integrate the sidebar with your main layout:

```tsx
import { SidebarProvider, SidebarInset } from '@/components/ui/sidebar'
import { AppSidebar } from '@/components/layout/app-sidebar'

export function Layout({ children }: { children: React.ReactNode }) {
  return (
    <SidebarProvider>
      <AppSidebar />
      <SidebarInset>
        <main className="flex flex-1 flex-col">
          {children}
        </main>
      </SidebarInset>
    </SidebarProvider>
  )
}
```

## Responsive Behavior

The sidebar automatically adapts to different screen sizes:

### Mobile
- Renders as an overlay/drawer
- Closes automatically after navigation
- Accessible via hamburger menu

### Desktop
- Visible by default
- Can be collapsed to icon-only view
- Persists state across navigation

### Implementation

From `src/components/layout/nav-group.tsx:36-58`:

```tsx
export function NavGroup({ title, items }: NavGroupProps) {
  const { state, isMobile } = useSidebar()
  const href = useLocation({ select: (location) => location.href })
  
  return (
    <SidebarGroup>
      <SidebarGroupLabel>{title}</SidebarGroupLabel>
      <SidebarMenu>
        {items.map((item) => {
          const key = `${item.title}-${item.url}`

          // Simple link for items without children
          if (!item.items)
            return <SidebarMenuLink key={key} item={item} href={href} />

          // Dropdown menu when collapsed on desktop
          if (state === 'collapsed' && !isMobile)
            return (
              <SidebarMenuCollapsedDropdown key={key} item={item} href={href} />
            )

          // Collapsible menu when expanded
          return <SidebarMenuCollapsible key={key} item={item} href={href} />
        })}
      </SidebarMenu>
    </SidebarGroup>
  )
}
```

## Active Link Highlighting

Navigation items automatically highlight based on the current route:

```tsx
function checkIsActive(href: string, item: NavItem, mainNav = false) {
  return (
    href === item.url || // Exact match
    href.split('?')[0] === item.url || // Match without query params
    !!item?.items?.filter((i) => i.url === href).length || // Child is active
    (mainNav &&
      href.split('/')[1] !== '' &&
      href.split('/')[1] === item?.url?.split('/')[1]) // Section match
  )
}
```

## Sidebar Hooks

### useSidebar

Access sidebar state and controls:

```tsx
import { useSidebar } from '@/components/ui/sidebar'

function Component() {
  const {
    state,        // 'expanded' | 'collapsed'
    open,         // boolean
    isMobile,     // boolean
    openMobile,   // boolean
    setOpenMobile,
    toggleSidebar,
  } = useSidebar()

  return (
    <button onClick={toggleSidebar}>
      {state === 'expanded' ? 'Collapse' : 'Expand'}
    </button>
  )
}
```
