## Get Started Quickly

<Steps>
  <Step title="Install Dependencies">
    Clone the repository and install dependencies with your preferred package manager
    
    ```bash
    git clone https://github.com/satnaing/shadcn-admin.git
    cd shadcn-admin
    pnpm install
    ```
  </Step>
  
  <Step title="Start Development Server">
    Run the development server and open your browser
    
    ```bash
    pnpm run dev
    ```
  </Step>
  
  <Step title="Customize Your Dashboard">
    Explore the components, pages, and customization options to build your admin interface
  </Step>
</Steps>
