The dashboard is the main landing page for authenticated users, featuring metrics cards, charts, and data visualization components.

## Overview

The dashboard provides a comprehensive view of key metrics and analytics through an interactive tabbed interface.

**Location:** `src/features/dashboard/index.tsx`  
**Route:** `/_authenticated/` (root authenticated route)

## Customization

### Adding New Metrics

To add a new metric card:

1. Add it to the metrics grid in the dashboard component
2. Use the same `Card` structure with `CardHeader` and `CardContent`
3. Include an icon SVG and metric values

```tsx
<Card>
  <CardHeader className='flex flex-row items-center justify-between space-y-0 pb-2'>
    <CardTitle className='text-sm font-medium'>Your Metric</CardTitle>
    {/* Add your icon */}
  </CardHeader>
  <CardContent>
    <div className='text-2xl font-bold'>Value</div>
    <p className='text-xs text-muted-foreground'>Description</p>
  </CardContent>
</Card>
```

### Adding New Tabs

To enable disabled tabs or add new ones:

1. Add/update the `TabsTrigger` in the tabs list
2. Create the corresponding `TabsContent` component
3. Build your custom content component

```tsx
<TabsTrigger value='reports'>Reports</TabsTrigger>

<TabsContent value='reports' className='space-y-4'>
  <YourReportsComponent />
</TabsContent>
```

### Modifying Top Navigation

Update the `topNav` array to change navigation links:

```tsx
const topNav = [
  {
    title: 'Your Link',
    href: 'path/to/page',
    isActive: false,
    disabled: false,
  },
  // ...
]
```

## Responsive Design

The dashboard is fully responsive:

- **Mobile:** Single column layout, stacked cards
- **Tablet:** 2 column grid for metrics, stacked content
- **Desktop:** 4 column metrics grid, side-by-side charts

<Info>
The layout uses Tailwind CSS responsive utilities (`sm:`, `lg:`) for breakpoint-based styling.
</Info>
